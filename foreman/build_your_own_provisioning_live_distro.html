#Cfgmgmtcamp

Config Management Camp

3 and 4 February 2014

Gent, Belgium

Build your own provisioning live distro

17:00 - 17:20 Lukas Zapletal

Foreman Discovery is a plugin which enables MaaS hardware hardware discovery in Foreman. Uknown hosts are booting into specialized stateless kernel over TFTP which acts as a pre-previsioning environment. It registers the machine to the Foreman, sends some hardware and network info and administrator is able to restart and start provisioning using single mouse click. The live kernel and image is based on oVirt Node project and in this hack session we will create our own custom live image, modify it via oVIrt Node plugin and test it in a VM.
