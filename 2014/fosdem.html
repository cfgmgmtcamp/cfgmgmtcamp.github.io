3 and 4 February 2014

Gent, Belgium

FOSDEM Configuration Management Devroom

The weekend before CfgMgmtCamp.eu the famous FOSDEM conference takes place in Brussels. At FOSDEM, there will be a devroom dedicated to configuration management. The Configuration Management Devroom is taking place on Saturday, February 1 in room S.H.1309 (Van Rijn), which is room 1309 in the H building and holds 146 people.

Agenda

  • 11:00 - 12:00: Configuration Management 101 - Sean O'Meara
  • 12:00 - 12:25: Use case: Configuration Management in the enterprise Linux environment. How I automated myself out of my job in 14 months - Remi Bergsma
  • 12:30 - 12:55: ncf abstracting CFEngine's complexity to provide a structured and powerful framework - Johnathan Clarke
  • 13:00 - 14:00: Lunch
  • 14:00 - 14:25: Introduction to Docker. Containerization is the new virtualization - James Turnbull
  • 14:30 - 14:55: A metadata ocean in Puppet and Chef. How to cope with metadata organisation - Marc Cluet
  • 15:00 - 15:25: SaltStack: Configuration Management Meets Remote Execution - Corey Quinn
  • 15:30 - 15:55: Razor: Provision like a Boss - David Lutterkort
  • 16:00 - 16:30: Break
  • 16:30 - 16:55: Foreman integration with Chef (and others) - Marek Hulan
  • 17:00 - 17:25: Manageable Puppet Infrastructure. Forging the pieces together - Ger Apeldoorn
  • 17:30 - 17:55: The classification problem: challenges and solutions. External node classification, the CFEngine way - Marco Marongiu
  • 18:00 - 18:25: NixOS: declarative configuration Linux distribution - Domen Kozar
  • 18:30 - 19:00: Service orchestration in the cloud with Juju - Marco Ceppi
