About Spencer Krum

Puppet performs configuration management, Ansible performs orchestration. In this talk, we'll talk about what prompted us to create this system, what we've been able to do with it, and where we are going next. We'll describe the setup of our systems and network. We'll also show you some things that we tried that didn't work, so that you don't have to make the same mistake. Everything you see in this talk is 100% Open Source and running live on the OpenStack Infrastructure servers.
